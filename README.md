# My Portfolio

A modern, interactive portfolio website built with React and Vite, showcasing skills, projects, and professional information.

## Features

- **Loading Screen**: Animated initial loading experience with smooth transitions
- **Interactive Particles Background**: Dynamic particle effects using react-particles
- **Custom Cursor**: Personalized cursor that follows mouse movement
- **Smooth Animations**: Powered by Framer Motion for fluid page transitions
- **Responsive Design**: Fully responsive layout that works on all devices
- **About Section**: Professional biography and introduction
- **Skills Section**: Display of technical skills and competencies
- **Projects Section**: Showcase of portfolio projects with descriptions
- **Contact Section**: Contact form and communication options
- **Footer**: Social links and copyright information

## Tech Stack

- React 18
- Vite
- Framer Motion
- React Particles
- CSS3

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
